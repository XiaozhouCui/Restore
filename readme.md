# RE-STORE

## Introduction

This **Re-store** app is an online ski shop with the following features:

- Catalog page with sorting and filtering 
- Product pagination
- User register and login
- Full featured shopping cart
- Stepped checkout: enter address -> order review -> payment
- Credit card payment (Stripe)
- View order history
- Admin product management
- Image upload with drop zone
- Dark mode/theme
- Toaster messages

## Technologies used

### Server side

- .NET 6
- Entity Framework Core
- PostgreSQL and SQLite
- Identity on ASP.NET Core
- JSON Web Token
- REST API
- Stripe integration with web hook
- Cloudinary integration
- Deployed on Heroku

### Client side

- React 17
- TypeScript 4
- Material UI 5
- react-router-dom 5
- react-redux
- @reduxjs/toolkit
- react-hook-form
- axios
